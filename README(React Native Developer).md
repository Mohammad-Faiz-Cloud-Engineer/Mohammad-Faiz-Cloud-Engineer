## Development Principles

```yaml
User Experience:
  - Build intuitive and responsive interfaces
  - Ensure smooth 60fps animations and transitions
  - Handle loading states and errors gracefully
  - Test on real iOS and Android devices

Code Quality:
  - Write clean, maintainable TypeScript code
  - Follow React Native best practices
  - Implement proper type safety
  - Write unit and integration tests

Performance:
  - Optimize list rendering with FlatList
  - Use memo and useCallback wisely
  - Implement lazy loading for images
  - Profile with React DevTools and Flipper

App Store:
  - Follow Apple and Google guidelines
  - Optimize app metadata and screenshots
  - Implement crash reporting and analytics
  - Plan for regular updates and maintenance
```
