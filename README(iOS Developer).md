## Development Principles

```yaml
User Experience:
  - Build intuitive and accessible interfaces
  - Follow Apple Human Interface Guidelines
  - Handle all edge cases gracefully
  - Test on multiple devices and iOS versions

Code Quality:
  - Write clean, idiomatic Swift
  - Follow iOS best practices
  - Implement proper error handling
  - Write unit and UI tests with XCTest

Performance:
  - Optimize app launch time
  - Use lazy loading for lists and images
  - Profile with Instruments
  - Implement proper memory management

Deployment:
  - Follow App Store Review Guidelines
  - Optimize app bundle size
  - Implement crash reporting with Crashlytics
  - Plan for regular updates
```
